HTML

  • The opening tag represents the root or start of our HTML document.
  • The head element contains information about the webpage.
  • Meta is a head element that contains metadata about our webpage to effectively serve our webpage to users.
  • The body element represents the visible content shown to the user.

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS box model consists of margins outside of an element and the element's borders, padding, and content.

Git

  • git status: checks what branch we are currently working on
  • git checkout -b branch name: creates a new branch and switches to it

JavaScript

  • JavaScript is case sensitive.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • We can use conditional statements and loops to control the oerder in which code is executed or if it's executed at all.
  • An array is a single variable that is used to hold a group of data.
  • An array index allows us to work with individual items inside an array.
  • Arrays are zero-indexed and sequential.
  • A "for" loop uses the predictable pattern of indices to perform a task on all items in an array by allowing a single code block to be executed over and over.
  • A "for" loop contains the following three important statements, the starting point for our loop, the condition, and the final statement.
  • A function is a set of instructions that tells the computer how to perform a certain task.
  • A function's name is followed by parentheses().
  • Declaring a function in our JavaScript file does not mean that the code will automatically execute. We need to call the function in the code at the spot we want it to execute.
